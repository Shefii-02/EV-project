
# EV route planner

Instantly find charging stations nearby



## Installation

Clone EV route planner

``` 
 git repo https://github.com/Shefii-02/EV-project.git
```

Install dependencies

``` 
  composer install
```
Run migration

``` 
  php artisan migrate
```

Start application

``` 
  php artisan serve
```
